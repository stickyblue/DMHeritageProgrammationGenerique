# DM: Héritage, Programmation Générique

Ce repository correspond au rendu du DM d'architecture et structure de données. Le sujet ne possède pas de question 17 au jour de cette rédaction (24/12/25).

## Compilation

La compilation se fera au travers d'un makefile utilisant g++ pour c++20

L'exécution du programme s'effectuera en tapant ./Exercice

## Vérification des fonctions

L'entièreté des fonctions seront vérifiée dans le fichier main.cpp par l'intermédiaire d'output console

## Réponses aux questions

Les questions ne necessitant pas de code seront placées dans un fichier Markdown à part, dans le dossier doc

## Méthodes complexes

Les méthodes complexes ainsi que la documentation des méthodes plus simples seront au besoin expliquées également dans un fichier Markdown présent dans le dossier doc

## Questions et troubleshouting

Les questions et / ou problèmes de compilation peuvent être adressées dans l'onglet issues du repo, ou par mail s'il s'agit d'un examinateur.
